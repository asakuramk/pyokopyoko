# お家へ帰ろう：ぴょこぴょこジャンプ 🏠

ブラウザで遊べる縦スクロール型ジャンプゲームです。

## 🎮 ゲーム概要

可愛いキャラクターを操作して、足場から足場へぴょこぴょこジャンプしながら、6000m上空にあるお家を目指すアクションゲームです。

## ✨ 特徴

- **3種類の足場**
  - 🟢 通常の足場（緑）- 普通にジャンプできる
  - 🔵 動く足場（青）- 左右に移動する
  - 🟤 壊れる足場（茶）- 一度乗ると壊れる
  
- **可愛いキャラクター**
  - 表情が変化（ジャンプ中・落下中・通常）
  - パーティクルエフェクト付き

- **段階的な難易度上昇**
  - 高さに応じて難易度がアップ
  - 1000mごとにレベルが上がる

## 🕹️ 操作方法

- **PC**: 矢印キー（← →）で左右移動
- **スマホ**: 画面の左右をタップ
- 足場に着地すると自動的にジャンプします

## 🔊 サウンド

効果音付き（ジャンプ、足場が壊れる、ゲームオーバー、クリア）
右上のボタンでミュート可能 🔊/🔇

## 遊び方

1. `index.html` をブラウザで開く
2. 「スタート！」ボタンをクリック
3. お家を目指してジャンプ！

## 技術

- HTML5 Canvas
- JavaScript
- Web Audio API
